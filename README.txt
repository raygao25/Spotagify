User manual:
	-Add tag: Type tag name in the top textbox and press "Add tag" button. Please note that
		Duplicate tag will not be added.
		
	-Delete tag: Type tag name and press "Delete tag" button.
	
	-Assign tag: Press "Manage tag" button which lies beside song name. Then check the tags
		you want to add, and press "Apply" button to finish.
	
	-Assign rating: Check the radio buttons below the song.
	
	-Filter by tags: Check the tags you want to filter with, and press "Match all" or "Match 
		any" button.
	
	-Filter by rating: Check the radio buttons below "Rating:" and press "Apply"
